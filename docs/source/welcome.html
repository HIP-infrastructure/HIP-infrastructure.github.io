Welcome to the HIP Documentation Portal

This portal provides comprehensive resources to help you navigate and use the Human Intracerebral EEG Platform. Whether you're just getting started or diving into advanced analyses, this site is your central reference.

Here, you will find

  • Onboarding Materials Step-by-step instructions to request access, log in, and begin working within the HIP
  • User Guides Detailed documentation on core platform features, including data upload, workspace management, and security practices
  • Tutorials Hands-on examples that walk you through typical workflows, from SEEG preprocessing to imaging coregistration and analysis pipelines
  • Available Datasets Descriptions and access conditions for curated datasets shared within the HIP environment
  • Applications A list of integrated applications available on the platform, organized by research focus

Access the platform at thehip.app
